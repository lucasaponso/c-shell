## C-SHELL

### A minimal shell program...


Supports:
- CSH-LANG (A custom scripting language)
- ENV Variables


### Manual Install
```
sudo install -Dm755 bin/cshell /usr/bin/cshell
```

#### Retrieve Latest Changes
```
git checkout release
git fetch
git pull
```

#### Build
```
mkdir -p build/
chmod +x build.sh
./build.sh
```
